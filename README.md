# Ardmore Cricket Club Website

This repository contains a simple static website for Ardmore Cricket Club designed to be hosted on GitHub Pages.

## Local development

Open `index.html` in your browser to preview the site. All assets are loaded from the `assets/` directory and external CDNs.

## Deployment

Commit changes to the `main` branch (or the branch configured for GitHub Pages) and push to GitHub. GitHub Pages will serve the latest version automatically.

## Customising

- Update club information, fixtures, or news articles by editing the corresponding sections in `index.html`.
- Adjust colours, typography, or layout via `assets/css/styles.css`.
- Replace the hero background image by updating the `background` declaration in the `.hero` rule within the stylesheet.

For questions or improvements please raise an issue or submit a pull request.
